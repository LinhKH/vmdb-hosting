## Required Environments
- PHP: 8.1.12
- Laravel: 9.47.0
- Node: 17.9.1
- NPM: 8.11.0
- Vuejs: 3.*

## How to set up development?
- Step 1: cd "{project/path}"
- Step 2: composer install
- Step 3: npm install
- Step 4: npm run dev
- Step 5: You can access admin site in your web browser at: http://localhost/laravel-app/public
